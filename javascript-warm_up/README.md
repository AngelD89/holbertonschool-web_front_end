# JavaScript - Warm up

This project contains JavaScript scripts for learning basic programming concepts.

## Learning Objectives

- Why JavaScript programming is amazing
- How to run a JavaScript script
- How to create variables and constants
- What are differences between var, const and let
- What are all the data types available in JavaScript
- How to use the if, if ... else statements
- How to use comments
- How to affect values to variables
- How to use while and for loops
- How to use break and continue statements
- What is a function and how do you use functions
- What does a function that does not use any return statement return
- Scope of variables
- What are the arithmetic operators and how to use them
- How to manipulate dictionary
- How to import a file

## Requirements

- Ubuntu 20.04 LTS using node (version 14.x)
- All files should end with a new line
- The first line of all files should be exactly `#!/usr/bin/node`
- Code should be semistandard compliant (version 16.x.x)
- All files must be executable

## Tasks

0. **0-javascript_is_amazing.js** - Prints "JavaScript is amazing"
1. **1-multi_languages.js** - Prints 3 lines
2. **2-arguments.js** - Prints a message depending on the number of arguments
3. **3-value_argument.js** - Prints the first argument passed to it
4. **4-concat.js** - Prints two arguments in a specific format
5. **5-to_integer.js** - Prints the first argument converted to an integer
6. **6-multi_languages_loop.js** - Prints 3 lines using a loop
7. **7-multi_c.js** - Prints "C is fun" x times
8. **8-square.js** - Prints a square
9. **9-add.js** - Prints the addition of 2 integers
10. **10-factorial.js** - Computes and prints a factorial
11. **11-second_biggest.js** - Searches the second biggest integer in the list
12. **12-object.js** - Updates a value in an object
13. **13-add.js** - Function that returns the addition of 2 integers
